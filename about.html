What is COVID-19?

    COVID-19 is a disease caused by the “novel corona virus”.
      Common symptoms are:
    • Fever
    • Dry cough
    • Breathing difficulty
    • Some patients also have aches and pains, nasal congestion, runny nose, sore throat.


    Here in this Project Temperature and Heart Rate is considered and local analysis of BP calculation is done to take local action as to allow a person or not. If it exceeds the threshold then data is recorder and screening is done on travel history. After COVID 19 testing data is updated in Database and Exploratory Data analysis is done. An exploratory data analysis with visualizations has been made to understand the number of different cases reported. The epidemiological data need to be analysed in a way so that the exploratory data analysis (EDA) methods and visualization model will increase the situational awareness.
